Funksiya

Funksiyani elon qilish

Funksiyani elon qilishda
1-(function) kalit sozi
2-Funksiya nomi
3-qavs ichida (parametr1, parametr2, ... parametrN)lar berish
4-Funksiya scop ichiga kerakli kodni yozamiz
5-Funksiyani chaqirish:>> Funksiya nomi (argument1, argument2,...argumentN)lar orqali chaqiriladi

Masalan:

function meningFunksiyam(son1,son2) {
if (son1 < son2) {
console.log(`${son2} ${son1}dan katta`);
} else {
console.log(`${son1} ${son2}dan katta`);
}
}
meningFunksiyam(Math.round(Math.random()*100),Math.round(Math.random()*100))

Funksiyani birir o'zgaruvchiga qiymat sifatida saqlab ishlatsa ham boladi Lekin bunay ishlatishda funksiyani elon qilishdan avval chaqirib bolmaydi

Arrov Funksiyasi (arrow function)(=>) bu funksiyani yozishni biroz oson usuli bolib bunda funksiya biron o'zgaruvchiga qiymat sifatida yoziladi va bunda (function) sozidan foydalanilmaydi

Masalan: 1-usul

let funksiya = (son) => {
return son +=20
}
console.log(funksiya(10));

Agar funksiya faqat bir parametrlik bo'lsa parametrdagi qavslarni yozmasa ham bo'ladi

let funksiya = son => {
return son +=20
}
console.log(funksiya(10));

Agar funksiyani tana qismi faqat bit qatorlik bo'lsa unga {} qavslar va tana qismini yozmasa ham bo'ladi

let funksiya = son => son+=10
console.log(funksiya(10));

Agar funksiya hech qanday parametrga ega bolmasa unda () qavslarni bo'sh qoldirb yozsa ham boladi

let funksiya = () => console.log("nma gap");
funksiya()

Agar tana qismi koproq bolsa

let funksiya = () => {
console.log("nma gap");
console.log(12);
}
funksiya()

Agar parametrlar 2 va undan kop bo'lsa parametrlar qavs ichiga yozilishi shart

let funksiya =(a,b,c) => {
console.log(a+b+c);
}
funksiya(1,2,3)
